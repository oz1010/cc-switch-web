# 1.2 インストールガイド

## 前提条件

### Node.js のインストール

CC Switch が管理する CLI ツール（Claude Code、Codex、Gemini CLI）には Node.js 環境が必要です。

**推奨バージョン**：Node.js 18 LTS 以上

#### Windows

1. [Node.js 公式サイト](https://nodejs.org/) にアクセス
2. LTS 版インストーラーをダウンロード
3. インストーラーを実行し、案内に従ってインストール
4. インストール確認：

```bash
node --version
npm --version
```

#### macOS

```bash
# Homebrew でインストール
brew install node

# または nvm を使用（推奨）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
```

#### Linux

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# または nvm を使用
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
```

### CLI ツールのインストール

#### Claude Code

**方法 1：Homebrew（macOS 推奨）**

```bash
brew install claude-code
```

**方法 2：npm**

```bash
npm install -g @anthropic-ai/claude-code
```

#### Codex

**方法 1：Homebrew（macOS 推奨）**

```bash
brew install codex
```

**方法 2：npm**

```bash
npm install -g @openai/codex
```

#### Gemini CLI

**方法 1：Homebrew（macOS 推奨）**

```bash
brew install gemini-cli
```

**方法 2：npm**

```bash
npm install -g @google/gemini-cli
```

## 正式リリース形式

GitHub Releases では Web ランタイムのみを正式公開します。

| プラットフォーム | アセット | 説明 |
| --- | --- | --- |
| Windows x86_64 | `cc-switch-web-v{version}-windows-x86_64.exe` | そのまま実行 |
| macOS x86_64 | `cc-switch-web-v{version}-macos-x86_64` | 初回実行前に `chmod +x` が必要 |
| Linux x86_64 | `cc-switch-web-v{version}-linux-x86_64-ubuntu20.04` | Ubuntu 20.04 上でビルド、互換ベースラインは Ubuntu 20.04+ |

既定のアクセス先：`http://127.0.0.1:17666`

すべての正式アセットで以下の環境変数を利用できます。

```bash
CC_SWITCH_PORT=8080
CC_SWITCH_HOST=0.0.0.0
CC_SWITCH_AUTH_TOKEN=your-secret
```

## Windows

1. [Releases ページ](https://github.com/farion1231/cc-switch/releases) を開く
2. `cc-switch-web-v{version}-windows-x86_64.exe` をダウンロード
3. ダウンロード先で PowerShell またはコマンドプロンプトを開く
4. 実行：

```powershell
.\cc-switch-web-v{version}-windows-x86_64.exe
```

5. `http://127.0.0.1:17666` を開く

## macOS

1. `cc-switch-web-v{version}-macos-x86_64` をダウンロード
2. ダウンロード先でターミナルを開く
3. 実行：

```bash
chmod +x ./cc-switch-web-v{version}-macos-x86_64
./cc-switch-web-v{version}-macos-x86_64
```

4. `http://127.0.0.1:17666` を開く

### Gatekeeper の注意

macOS バイナリは現在未署名です。Gatekeeper にブロックされた場合は、隔離属性を削除してから再実行してください。

```bash
xattr -d com.apple.quarantine ./cc-switch-web-v{version}-macos-x86_64
```

## Linux

1. `cc-switch-web-v{version}-linux-x86_64-ubuntu20.04` をダウンロード
2. ダウンロード先でターミナルを開く
3. 実行：

```bash
chmod +x ./cc-switch-web-v{version}-linux-x86_64-ubuntu20.04
./cc-switch-web-v{version}-linux-x86_64-ubuntu20.04
```

4. `http://127.0.0.1:17666` を開く

### 互換性メモ

正式な Linux バイナリは Ubuntu 20.04 上でビルドされます。文書上の最低ターゲット環境は Ubuntu 20.04 または互換ディストリビューションです。

## ランタイム確認

起動後、次を確認してください。

1. プロセスが正常に起動し、クラッシュしていない
2. `http://127.0.0.1:17666` で Web UI が表示される
3. `CC_SWITCH_PORT` と `CC_SWITCH_HOST` の上書きが期待どおりに動作する

## 停止方法

このバイナリを起動したターミナルで `Ctrl+C` を押してください。
