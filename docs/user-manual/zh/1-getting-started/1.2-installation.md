# 1.2 安装指南

## 前置要求

### 安装 Node.js

CC Switch 管理的 CLI 工具（Claude Code、Codex、Gemini CLI）需要 Node.js 环境。

**推荐版本**：Node.js 18 LTS 或更高版本

#### Windows

1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 LTS 版本安装包
3. 运行安装程序，按提示完成安装
4. 验证安装：

```bash
node --version
npm --version
```

#### macOS

```bash
# 使用 Homebrew 安装
brew install node

# 或使用 nvm（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
```

#### Linux

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# 或使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
```

### 安装 CLI 工具

#### Claude Code

**方式一：Homebrew（macOS 推荐）**

```bash
brew install claude-code
```

**方式二：npm**

```bash
npm install -g @anthropic-ai/claude-code
```

#### Codex

**方式一：Homebrew（macOS 推荐）**

```bash
brew install codex
```

**方式二：npm**

```bash
npm install -g @openai/codex
```

#### Gemini CLI

**方式一：Homebrew（macOS 推荐）**

```bash
brew install gemini-cli
```

**方式二：npm**

```bash
npm install -g @google/gemini-cli
```

## 正式发布形式

GitHub Releases 仅正式发布 Web 运行版本。

| 平台 | 资产 | 说明 |
| --- | --- | --- |
| Windows x86_64 | `cc-switch-web-v{version}-windows-x86_64.exe` | 直接运行 |
| macOS x86_64 | `cc-switch-web-v{version}-macos-x86_64` | 首次运行前需要 `chmod +x` |
| Linux x86_64 | `cc-switch-web-v{version}-linux-x86_64-ubuntu20.04` | 在 Ubuntu 20.04 上构建，兼容基线为 Ubuntu 20.04+ |

默认访问地址：`http://127.0.0.1:17666`

所有正式资产支持以下环境变量：

```bash
CC_SWITCH_PORT=8080
CC_SWITCH_HOST=0.0.0.0
CC_SWITCH_AUTH_TOKEN=your-secret
```

## Windows

1. 访问 [Releases 页面](https://github.com/farion1231/cc-switch/releases)
2. 下载 `cc-switch-web-v{version}-windows-x86_64.exe`
3. 在下载目录中打开 PowerShell 或命令提示符
4. 运行：

```powershell
.\cc-switch-web-v{version}-windows-x86_64.exe
```

5. 打开 `http://127.0.0.1:17666`

## macOS

1. 下载 `cc-switch-web-v{version}-macos-x86_64`
2. 在下载目录中打开终端
3. 运行：

```bash
chmod +x ./cc-switch-web-v{version}-macos-x86_64
./cc-switch-web-v{version}-macos-x86_64
```

4. 打开 `http://127.0.0.1:17666`

### Gatekeeper 提示

macOS 二进制当前未签名。如果 Gatekeeper 阻止启动，请先移除隔离属性后再运行：

```bash
xattr -d com.apple.quarantine ./cc-switch-web-v{version}-macos-x86_64
```

## Linux

1. 下载 `cc-switch-web-v{version}-linux-x86_64-ubuntu20.04`
2. 在下载目录中打开终端
3. 运行：

```bash
chmod +x ./cc-switch-web-v{version}-linux-x86_64-ubuntu20.04
./cc-switch-web-v{version}-linux-x86_64-ubuntu20.04
```

4. 打开 `http://127.0.0.1:17666`

### 兼容性说明

正式 Linux 二进制在 Ubuntu 20.04 上构建，文档约定的最低目标环境为 Ubuntu 20.04 或兼容发行版。

## 验证运行时

启动后，请确认：

1. 进程正常启动且没有崩溃
2. `http://127.0.0.1:17666` 能访问 Web UI
3. `CC_SWITCH_PORT` 和 `CC_SWITCH_HOST` 覆盖项行为符合预期

## 停止服务

在启动该二进制的终端中按 `Ctrl+C` 即可停止。
